# Chocolate-Sales-SQL-Analysis-Project
This project presents a complete SQL-based business analysis of a chocolate sales dataset. The goal is to analyze revenue performance, product effectiveness, country contribution, and salesperson performance using SQL queries.  The analysis answers key business questions that help drive strategic decision-making.

| Column Name   | Data Type     | Description             |
| ------------- | ------------- | ----------------------- |
| sale_id       | SERIAL (PK)   | Unique sale identifier  |
| sales_person  | VARCHAR(100)  | Name of salesperson     |
| country       | VARCHAR(100)  | Country of sale         |
| product       | VARCHAR(100)  | Product sold            |
| sale_date     | DATE          | Transaction date        |
| amount        | NUMERIC(12,2) | Revenue generated       |
| boxes_shipped | INTEGER       | Number of boxes shipped |
